Tutorial - Query IP Conflict

若您的IP被人佔用，可依照下列步驟找出對方卡號

  • 命令列模式
    • Windows 98：開始 > 執行 > 鍵入[ command ] > 鍵入[ nbtstat -A 您被佔用的IP ]
    • Windows XP：開始 > 執行 > 鍵入[ cmd ] > 鍵入[ nbtstat -A 您被佔用的IP ]
    • 如下圖所示(被佔用IP時會有兩張卡號，請不要抄到自己的)：
  • Windows 2K/XP專用模式
    • 控制台 > 系統管理工具 > 事件檢視器 > 系統
    • 找「類型為錯誤」且「來源為Tcpip」的紀錄
